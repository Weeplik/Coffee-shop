# Coffee Shop — Landing Page

Одностраничный сайт (Landing Page) для кофейни **«Coffee Shop»**, разработанный в соответствии с техническим заданием.

---

## Быстрый старт

Проект не требует сборки и серверных технологий. Достаточно открыть `index.html` в браузере:

```bash
# Вариант 1 — просто открыть файл
open index.html          # macOS
xdg-open index.html      # Linux
start index.html         # Windows

# Вариант 2 — локальный сервер (рекомендуется, чтобы корректно загружались шрифты)
npx serve .
# или
python3 -m http.server 8080
# затем перейти на http://localhost:8080
```

## JavaScript-функциональность

- **Мобильное меню** — бургер-кнопка с анимацией открытия/закрытия
- **Хедер при скролле** — тень и размытие фона при прокрутке
- **Активные ссылки** — подсветка текущей секции через `IntersectionObserver`
- **Фильтрация меню** — показ карточек по категориям (Кофе / Десерты / Чай)
- **Форма бронирования** — клиентская валидация + toast-уведомление
- **Кнопка «Наверх»** — появляется после прокрутки 400px
- **Анимации появления** — плавный вход элементов при скролле
